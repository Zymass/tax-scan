# Как посмотреть логи backend сервера

## Вариант 1: Перезапустить сервер в своем терминале (рекомендуется)

1. Откройте терминал (Terminal.app на Mac)
2. Перейдите в директорию backend:
   ```bash
   cd /Users/iliafiliaev/Develop/TaxCalculator/backend
   ```
3. Остановите текущий процесс (если нужно):
   ```bash
   # Найдите процесс
   ps aux | grep "tsx watch"
   # Остановите его (замените PID на реальный)
   kill <PID>
   ```
4. Запустите сервер заново:
   ```bash
   npm run dev
   ```
5. Теперь все логи будут видны в этом терминале!

## Вариант 2: Посмотреть логи запущенного процесса

Если процесс уже запущен, можно попробовать найти его вывод:

```bash
# Найти процесс
ps aux | grep "tsx watch"

# Посмотреть, куда идет вывод (обычно это терминал, где был запущен)
```

## Вариант 3: Добавить логирование в файл

Можно настроить запись логов в файл, но для этого нужно изменить код.

## Быстрая проверка - работает ли сервер:

```bash
curl http://localhost:3000/health
```

Если видите `{"status":"ok",...}` - сервер работает!

## Когда попробуете скачать PDF:

После перезапуска в терминале вы увидите:
```
[PDF] Starting PDF generation for calculation ...
[PDF Generator] Launching Puppeteer...
```

И если будет ошибка:
```
❌ [2024-...] Error 500: PDF generation failed: ...
```
